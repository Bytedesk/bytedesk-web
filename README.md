# 萝卜丝 · IM + 云客服

- 致力于提供稳定、可扩展、定制化的客户服务一站式平台
- [官方网站](https://www.bytedesk.com)

## 准备工作

- [注册账号](https://www.bytedesk.com/antv/user/login)
- [登录后台](https://www.bytedesk.com/admin#/login)
- 分配应用：登录后台->客服->渠道->添加Web
