ByteDesk Web Global Demo

This demo shows how to use ByteDesk Web in global format.

Installation


// Copy this code to your page
<script src="path/to/bytedesk-web.js"></script>
<script>
  const config = {
    theme: {
      primaryColor: '#2e88ff',
      position: 'right'
    },
    window: {
      title: 'Customer Service',
      width: 380,
      height: 640
    },
    chatParams: {
      org: 'your_org_id',
      t: '2',
      sid: 'your_sid'
    }
  };

  const bytedesk = new BytedeskWeb(config);
  bytedesk.init();
</script>
