安装步骤

1. 通过 CDN 引入

<script src="https://unpkg.com/bytedesk-web"></script>

2. 配置参数

const config = {
  baseUrl: 'https://ai.bytedesk.com',
  placement: 'bottom-right',
  marginBottom: 20,
  marginSide: 20,
  chatConfig: {
    org: 'your_org_id',  // 替换为您的组织ID
    t: "2",
    sid: 'your_sid'      // 替换为您的SID
  }
};

3. 初始化

const bytedesk = new BytedeskWeb(config);
bytedesk.init();

4. 使用方法

<button onclick="bytedesk.showChat()">打开聊天</button>
<button onclick="bytedesk.hideChat()">关闭聊天</button>

5. 可用方法

  • bytedesk.showChat() - 显示聊天窗口
  • bytedesk.hideChat() - 隐藏聊天窗口
