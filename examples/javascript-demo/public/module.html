ByteDesk Web Module Demo

This demo shows how to use ByteDesk Web in module format.

Installation


// Copy this code to your page
<script type="module">
  import { BytedeskWeb } from 'bytedesk-web';

  const config = {
    theme: {
      primaryColor: '#2e88ff',
      position: 'right'
    },
    window: {
      title: 'Customer Service',
      width: 380,
      height: 640
    },
    chatParams: {
      org: 'your_org_id',
      t: '2',
      sid: 'your_sid'
    }
  };

  const bytedesk = new BytedeskWeb(config);
  bytedesk.init();
</script>
